# Описание

Вы хотите планировать покупки? У вас портятся овощи и другие
продукты, которые вы так хотели попробовать? Не беда! С этим
приложением все изменится!

Теперь вы можете сформировать списки по рецептам и прикинуть
сколько где порций и какая у них калорийность

# Планы и цели

## Версия 1.0

Управление рецептами и списком покупок в рамках локальной сессии и сохранением в LocalStorage + CI\CD

### Сделано
- Добавить скрипт для простого разворота в рамках сервера с настройками в Nginx
- Добавить возможность управлять рецептами
- Добавить возможность управлять списком покупок отдельно от рецепта
- Добавление сохранения локального состояния в LocalStorage
- Реализовать работу с двумя сущностями Recipe и ShoppingItem (нужно для корректной работы с беком в дальнейшем и более прозрачной работы со списком покупок)
- Добавить возможность восстанавливать состояние из LocalStorage
- Настроен запуск e2e тестов
- Перейти на числовые поля там где это нужно
- Ребрендинг на новый репозиторий

### Планируется
- Реализовать подсчет количества определенного ShoppingItem (унификация единиц измерения)
- Возможность расшаривания списка между клиентами 
- Переход к WPA 
- Тесты для версии
 = Базовая проверка +
 = Проверка формы рецепта +
 = Проверка формы товара
 = Проверка отображения и изменения тестов

### Баги
- При выборе рецепта из списка нужно менять вкладку на Мои рецепты

- При открытии создания новой покупки она заполнена старыми данными

- Не отображается список рецептов после перехода на новые сущности

- Есть проблема с обновлением счетчиков на панели с рецептами

- Мобильная верстка в печали, использовать приложение в таком виде невозможно
